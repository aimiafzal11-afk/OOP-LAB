# OOP-LAB
This repository contains Object-Oriented Programming (OOP) lab programs implemented in C++.
It covers fundamental and advanced OOP concepts through practical examples and problem-solving exercises.

## Concepts Covered:
- Classes and Objects
- Constructors and Destructors
- Encapsulation
- Abstraction
- Inheritance
- Polymorphism
- Function Overloading
- Operator Overloading

## Purpose:
The purpose of this repository is to strengthen understanding of OOP principles using C++ and to serve as a reference for lab practice and academic assignments.

## Language Used:
C++

## Structure:
Each program is organized based on lab exercises and demonstrates specific OOP concepts with clear implementation.
